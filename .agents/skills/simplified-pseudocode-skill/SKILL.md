# Simplified Pseudocode Skill

Use this skill to produce a readable pseudocode view of the current implementation.

## Core goals

1. Preserve real control flow order.
2. Keep key conditions and exits intact.
3. Remove language-specific noise.

## Workflow

1. Identify primary function/branch scope.
2. Convert statements into concise steps.
3. Preserve guards, loops, and error exits.
4. Add minimal notes for non-obvious transitions.

## Output structure

Always include:

1. Simplified pseudocode
2. Branch notes
3. Uncertainty
4. Source references
