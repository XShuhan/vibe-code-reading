# Performance Considerations Skill

Use this skill to identify practical performance risks and optimization opportunities.

## Core goals

1. Identify dominant cost paths.
2. Explain complexity and resource pressure.
3. Suggest realistic optimization options.

## Workflow

1. Locate expensive loops/calls/IO.
2. Estimate relative complexity and scaling risk.
3. Separate evidence-backed facts from inference.
4. Suggest incremental improvements with tradeoffs.

## Output structure

Use compact sections:

- Hot paths
- Complexity notes
- Resource risks
- Optimization options
- Source references
